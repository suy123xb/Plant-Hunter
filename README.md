## 🎮 项目介绍

《植物猎人》是一个创新的儿童友好型公园设计项目，通过Web技术将设计方案转化为沉浸式游戏体验。

**四大主题场景：**
- 🌲 **森之探幽** - 跟随森林精灵吉米探索森林迷宫
- 🌾 **草之游野** - 与草原精灵米拉在草坡上自由奔跑
- 💧 **水之栖歌** - 随水之精灵凯特探索水生生态
- 🏜️ **沙之乐觅** - 跟沙漠精灵威尔穿越沙漠世界

---
## 🎮 核心玩法机制

### 游戏框架：四大生态区 × AI精灵 × 拍照闯关

《植物猎人》将整个公园划分为**四大生态主题区域**，每个区域由一位**AI植物精灵**守护，拥有独特的生态环境和植物种类。玩家需要完成一系列拍照打卡任务，解锁区域内的所有植物，最终成为真正的"植物猎人"。

---

### 🌲 四大生态区 & AI精灵

#### 1. 森之探幽 - 森林生态区
**守护精灵：吉米（Jimmy）**
- 🌳 **生态特征**：密林、高大乔木、林下植物
- 🍃 **代表植物**：白皮松、元宝枫、银杏、白蜡等
- 🎯 **关卡数量**：20种森林植物
- 💡 **教育重点**：树木结构、四季变化、森林生态系统

**精灵能力**：吉米能通过水晶球与大树对话，了解每棵树的故事和特征

#### 2. 草之游野 - 草原生态区
**守护精灵：米拉（Mila）**
- 🌾 **生态特征**：开阔草地、草本植物、花境
- 🌸 **代表植物**：狼尾草、细叶芒、柳叶马鞭草、薰衣草等
- 🎯 **关卡数量**：20种草本植物
- 💡 **教育重点**：草本植物特征、花期观察、传粉昆虫

**精灵能力**：米拉能控制草地生长，展现植物的生命力

#### 3. 水之栖歌 - 水域生态区
**守护精灵：凯特（Kate）**
- 💧 **生态特征**：池塘、湿地、水生植物
- 🪷 **代表植物**：荷花、睡莲、芦苇、千屈菜等
- 🎯 **关卡数量**：20种水生植物
- 💡 **教育重点**：水生适应性、生态净化、水循环

**精灵能力**：凯特的发光苔藓瓶能吸引水生生物，帮助观察水下世界

#### 4. 沙之乐觅 - 旱生植物区
**守护精灵：威尔（Will）**
- 🏜️ **生态特征**：沙地、砾石、耐旱植物
- 🌵 **代表植物**：沙地柏、沙棘、仙人掌、景天等
- 🎯 **关卡数量**：20种耐旱植物
- 💡 **教育重点**：植物抗旱适应、节水生态、沙地保护

**精灵能力**：威尔能穿透沙尘，在恶劣环境中找到隐藏的植物

### 📸 拍照打卡闯关系统

#### 玩法流程：

```
进入生态区 → AI精灵出题 → 寻找植物 → 拍照上传 → AI识别 → 闯关成功 → 解锁下一关
```

#### 详细步骤：

**Step 1: 接受挑战**
- 进入某个生态区（如森林区）
- AI精灵吉米出现，介绍本区域特点
- 开启第1关挑战

**Step 2: 植物谜题**
- 精灵描述一种植物的特征，例如：
  > "我有银白色的树皮，像穿了一身铠甲。秋天时，我的叶子会变成金黄色的扇形。人们常用我的果实入药。我是谁？"

**Step 3: 实地寻找**
- 根据线索在公园内寻找对应植物
- 可以观察树叶形状、树皮纹理、生长位置等
- 家长可以协助，但鼓励孩子自主探索

**Step 4: 拍照上传**
- 找到后拍摄清晰照片
- APP内上传照片
- 系统提示"识别中..."

**Step 5: AI智能识别**
- AI精灵分析照片中的植物特征
- 匹配植物数据库
- 给出识别结果

**Step 6: 闯关反馈**
```
✅ 识别成功：
   "太棒了！这就是银杏树！你真是一位优秀的植物猎人！"
   → 获得积分和徽章
   → 解锁植物图鉴
   → 开启下一关

❌ 识别错误：
   "这不是我要找的植物哦，再仔细观察一下它的叶子形状吧！"
   → 给出提示
   → 可重新拍照
```

**Step 7: 知识解锁**
- 闯关成功后，精灵讲解植物知识：
  - 学名、别名、科属
  - 形态特征、生长习性
  - 生态价值、文化内涵
  - 有趣的植物故事

**Step 8: 连续闯关**
- 完成1关，解锁第2关
- 逐步探索该区域的20种植物
- 完成全部20关，该区域通关
