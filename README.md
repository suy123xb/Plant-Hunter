## 🎮 项目介绍

《植物猎人》是一个创新的儿童友好型公园设计项目，通过Web技术将设计方案转化为沉浸式游戏体验。

**四大主题场景：**
- 🌲 **森之探幽** - 跟随森林精灵吉米探索森林迷宫
- 🌾 **草之游野** - 与草原精灵米拉在草坡上自由奔跑
- 💧 **水之栖歌** - 随水之精灵凯特探索水生生态
- 🏜️ **沙之乐觅** - 跟沙漠精灵威尔穿越沙漠世界

---

## ✨ 特色功能

- 📱 **全平台适配** - 完美支持桌面、平板、手机
- 🎨 **精美视觉** - 手绘风格场景插画
- 🎵 **音效系统** - 沉浸式背景音乐和场景音效
- 🎯 **游戏化设计** - 任务闯关、积分奖励
- 💾 **进度保存** - 自动记录游玩进度

### 本地运行
```bash
git clone https://github.com/你的用户名/plant-hunter.git
cd plant-hunter
python -m http.server 8000
# 访问 http://localhost:8000
```
